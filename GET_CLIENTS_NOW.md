# SwarmOS — Get Clients Now (7-Day Plan)

## Goal: 1-5 paying clients, $1K-$5K revenue in 7 days

---

## Core Idea

You DON'T need:
- Perfect product
- Fancy UI

You NEED:
- Conversations -> Calls -> Closes

---

## Day 1 - Setup

### 1. Pick ONE niche
- SaaS founders
- Agencies
- Coaches

Example: "Marketing agencies doing $5K-$50K/month"

### 2. Your offer
"We book meetings for you automatically"

### 3. Landing page
Already built! Just deploy it.

---

## Day 2-7 - Daily Execution Loop

### Step 1: Get Leads (1-2 hours)
- Target: 100-200 leads/day
- Where: LinkedIn search, Apollo, Google

### Step 2: Send Messages (2-3 hours)
- Target: 100-300 messages/day

**Message template:**
```
Hey - quick question:

are you getting enough meetings consistently right now?

We've been helping similar companies automate this with AI.

Worth a quick look?
```

### Step 3: Handle Replies (Live)

**If interested:**
```
Nice - easiest is to show you.
Here's my calendar: [link]
```

**If unsure:**
```
Totally fair - quick question:
how are you currently getting clients?
```

Move them to a call ASAP.

### Step 4: Take Calls
Use the call script (SALES_SCRIPT.md). Close 20-30%.

### Step 5: Close
```
"We set everything up for you - it's $1K/month"
"Want me to get this live for you?"
```
